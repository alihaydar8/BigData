Une requête sur Wikidata

Thème : Country

La requête suivante retourne 100 résultats à la date du 17 janvier 2023.

https://query.wikidata.org/

Code de la requête : 

        
            SELECT ?country ?countryLabel ?instance_of ?instance_ofLabel ?image ?part_of ?part_ofLabel ?author_name_string WHERE {
                SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                ?country wdt:P31 wd:Q6256.
                OPTIONAL { ?country wdt:P31 ?instance_of. }
                OPTIONAL { ?country wdt:P18 ?image. }
                OPTIONAL { ?country wdt:P361 ?part_of. }
                OPTIONAL { ?country wdt:P2093 ?author_name_string. }
              }
              LIMIT 100
